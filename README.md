# Personal Finance & Expense Analytics App

A simple React app built with Vite for tracking income, expenses, budgets, and analytics.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`

## Features

- Add income and expense transactions
- Edit and delete transactions
- Search, filter, and sort transactions
- Monthly budget tracking
- Analytics dashboard with charts
- Simple UI built with React Router, React Hook Form, Yup, Recharts, and Axios
